# Paragons: The Roleplaying Game — Foundry VTT System

An unofficial Foundry VTT system for [Paragons: The Roleplaying Game](https://brokendoor.shop/paragons) by Broken Door Entertainment.

> This system is fan-made and not affiliated with or endorsed by Broken Door Entertainment.

## Installation

### Via Manifest URL (recommended)

In Foundry VTT, go to **Game Systems → Install System** and paste:

```
https://raw.githubusercontent.com/blackbeard718/paragons-fvtt/main/system.json
```

### Via The Forge

In your Forge-hosted Foundry instance, go to **Settings → Game Systems → Install System** and paste the manifest URL above.

## Features

- Full character sheet matching the official print layout
- NPC sheet with power rating guidelines and attack move roller
- Dice pool system — d6 pools with success (5) and critical (6=2 successes) counting
- Shift-click any stat or roll button for the full pre-roll dialog
- Cool Point spend button on roll chat cards — awards GM a Story Point automatically
- Reputation track (Villain → Paragon) with feat/failure counters
- Ability, Talent, and Gear item sheets with full mechanical fields
- Compendium packs for all abilities, archetype talents, unique gear, and sample NPCs
